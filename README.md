## Deskripsi

Program Sistem Manajemen Gudang dibuat menggunakan bahasa C++ untuk memenuhi tugas UAP Struktur Data dan Algoritma.

Program ini menerapkan beberapa konsep struktur data seperti:

## Materi yang Diterapkan

1. Struct

Digunakan untuk menyimpan data barang yang terdiri dari ID, nama barang, dan stok.

2. Pointer

Digunakan untuk menghubungkan node pada Linked List.

3. Linked List

Digunakan sebagai struktur dasar dalam implementasi Queue dan Stack.

4. Dynamic Memory Allocation

Menggunakan keyword new dan delete untuk membuat dan menghapus node secara dinamis.

5. Queue (FIFO)

Digunakan untuk menyimpan data barang di gudang. Barang yang masuk lebih dahulu akan keluar lebih dahulu.

6. Stack (LIFO)

Digunakan untuk menyimpan riwayat barang keluar. Riwayat terakhir dapat dihapus terlebih dahulu.

7. Object Oriented Programming (OOP)

Program menggunakan class Queue dan Stack untuk mengelola data.

8. Sorting

Digunakan untuk mengurutkan data barang berdasarkan stok dan nama barang.

9. Function

Setiap operasi seperti enqueue, dequeue, push, pop, dan sorting dibuat dalam fungsi tersendiri.

10. Switch Case

Digunakan untuk memilih menu yang dijalankan pengguna.

11. Perulangan (Do While)

Digunakan agar menu terus berjalan sampai pengguna memilih keluar dari program.

## Fitur Program

### 1. Tambah Barang

Menambahkan data barang ke dalam gudang.

### 2. Barang Keluar

Mengeluarkan barang berdasarkan konsep Queue (FIFO).

### 3. Tampilkan Data Gudang

Menampilkan seluruh data barang yang tersedia di gudang.

### 4. Tampilkan Riwayat

Menampilkan riwayat barang keluar menggunakan Stack.

### 5. Hapus Riwayat Terakhir

Menghapus riwayat transaksi terakhir berdasarkan konsep LIFO.

### 6. Sorting Stok Ascending

Mengurutkan data barang berdasarkan stok dari yang terkecil ke terbesar.

### 7. Sorting Nama A-Z

Mengurutkan data barang berdasarkan nama secara alfabetis.

## Struktur Data yang Digunakan

* Queue : Menyimpan data barang di gudang.
* Stack : Menyimpan riwayat barang keluar.
* Linked List : Implementasi dinamis untuk Queue dan Stack.
