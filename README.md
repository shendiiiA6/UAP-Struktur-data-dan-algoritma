## Deskripsi

Program Sistem Manajemen Gudang dibuat menggunakan bahasa C++ untuk memenuhi tugas UAP Struktur Data dan Algoritma.

Program ini menerapkan beberapa konsep struktur data seperti:

* Queue (FIFO)
* Stack (LIFO)
* Linked List
* Sorting
* Struct
* Pointer
* Object Oriented Programming (OOP)

## Fitur Program

### 1. Tambah Barang

Menambahkan data barang ke dalam gudang.

### 2. Barang Keluar

Mengeluarkan barang berdasarkan konsep Queue (FIFO).

### 3. Tampilkan Data Gudang

Menampilkan seluruh data barang yang tersedia di gudang.

### 4. Tampilkan Riwayat

Menampilkan riwayat barang keluar menggunakan Stack.

### 5. Hapus Riwayat Terakhir

Menghapus riwayat transaksi terakhir berdasarkan konsep LIFO.

### 6. Sorting Stok Ascending

Mengurutkan data barang berdasarkan stok dari yang terkecil ke terbesar.

### 7. Sorting Nama A-Z

Mengurutkan data barang berdasarkan nama secara alfabetis.

## Struktur Data yang Digunakan

* Queue : Menyimpan data barang di gudang.
* Stack : Menyimpan riwayat barang keluar.
* Linked List : Implementasi dinamis untuk Queue dan Stack.
